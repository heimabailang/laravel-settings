## Installing

```shell
$ composer require hongyukeji/laravel-settings
```

```shell
$ php artisan vendor:publish --provider='Hongyukeji\LaravelSettings\SettingsServiceProvider'
```

```shell
$ php artisan php artisan settings:create-table
```

```shell
$ php artisan migrate
```
